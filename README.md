# cat-rest

## 公開URL
```
https://cat-rest.com
```

## 概要
```
猫のTweetを収集して表示するWebサイトです。
猫種ごとに表示できます。

```

## 詳細
```
ソースはフロント側のみです。
Tweet自体はLambda上でTwitter APIを使って取得しており、LambdaからS3に保存した猫のtweet情報をVueで参照しています。
猫種はハッシュタグで判断しています。
PWAに対応しています。

```
